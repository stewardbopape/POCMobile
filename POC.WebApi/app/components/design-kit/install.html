Overview

The Stats SA Design System is a responsive and multipurpose admin theme powered with Twitter Bootstrap 3.3.7 Framework. The Design System can be used for any type of web applications: custom admin panels, admin dashboards, CMS, CRM, SAAS and more. The Design System has a sleek, clean and intuitive metro & flat balanced design which makes your next project look awesome and yet user friendly. The Design System has a huge collection of plugins and UI components and works seamlessly on all major web browsers, tablets and phones.

We put a lot of love and effort to make The Design System a useful tool for everyone and now The Design System comes with 7 complete admin layouts. We are keen to release continuous long term updates and dozens of new features will be coming soon in the future releases. Once you purchased The Design System, you will be entitled to free download of all future updates for the same license.
